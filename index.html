<p style="font-size:16px"><strong>The 10-K Corpus (extended version)</strong></p>
The 10-K reports from publicly traded U.S. companies from <b>year 1996
to 2013</b>:
<ul>
<li>
<b>Stock return volatility</b> measurements in the twelve-month
period before and after each report
</li>
<li>
<b>Abnormal trading volumes</b> (see [Loughran and McDonald 2011]
for the detailed definition)
</li>
<li>
Download (<a href="./static/README.txt">README</a>):
<ul>
<li>original 10-K reports <a href="./static/data/download/all.full.tgz">[all.full.tgz]</a> (10GB)</li>
<li>MD&A sections <a href="./static/data/download/all.mda.tgz">[all.mda.tgz]</a> (760MB)</li>
<li>tokenized MD&A sections <a href="./static/data/download/all.tok.tgz">[all.tok.tgz]</a> (533MB)</li>
<li>log postevent return volatility <a href="./static/data/download/all.logfama.tgz">[all.logfama.tgz]</a> (429KB)</li>
<li>log stock volatility <a href="./static/data/download/all.logvol.tgz">[all.logvol.tgz]</a> (800KB)</li>
<li>abnormal trading volumes <a href="./static/data/download/all.abnormal.tgz">[all.abnormal.tgz]</a>(429K)</li>
<li>excess returns <a href="./static/data/download/all.excess.tgz">[all.excess.tgz]</a>(405K)</li>
<li>meta files <a href="./static/data/download/all.meta.tgz">[all.meta.tgz]</a> (1.2MB)</li>
</ul>
</li>
</ul>
<hr>
<p style="font-size:16px"><strong>Pre-Trained Word Embeddings Vectors</strong></p>
We are publishing pre-trained vectors via <a
href="https://code.google.com/p/word2vec/"
target=_blank>word2vec</a> (with the CBOW model) trained on the
above 10-K Corpus (40,708 reports from 18 years). The models (w/ and w/o
incorporating syntactic information) contain 200-dimensional vectors
for each word.
<ul>
<li>
Pre-trained word vectors w/o incorporating syntactic information (359M tokens, 0.12M vocab, 200d vectors)
<ul>
<li>word vectors [<a href="./static/data/download/sim.expand.200d.vec.tgz">sim.expand.200d.vec.tgz</a>] (47MB)</li>
<li>bin file [<a href="./static/data/download/sim.expand.200d.bin.tgz">sim.expand.200d.bin.tgz</a>] (51MB) </li>
</ul>
</li>
<li>
Pre-trained word vectors w/ incorporating syntactic information (359M tokens, 0.26M vocab, 200d vectors)
<ul>
<li>word vectors [<a href="./static/data/download/syn.expand.200d.vec.tgz">syn.expand.200d.vec.tgz</a>] (83MB)</li>
<li>bin file [<a href="./static/data/download/syn.expand.200d.bin.tgz">syn.expand.200d.bin.tgz</a>] (90MB)</li>
</ul>
</li>
</ul>
<hr>

Please cite the first paper (ACM TMIS) if you write any papers involving the use of the data above:
<ul>
<li>
Ming-Feng Tsai, Chuan-Ju Wang, and Po-Chuan Chien. <b>Discovering Finance Keywords via Continuous Space Language Models.</b>
To appear in <i>ACM Transactions on Management Information Systems</i> (ACM TMIS).
</li>
<li>
Ming-Feng Tsai and Chuan-Ju Wang. <b>Financial Keyword Expansion via Continuous Word Vector Representations.</b>
<i>
Conference on Empirical Methods in Natural Language Processing</i>
(EMNLP '14), Doha, 2014, pp. 1453-1458. [<a href="http://www.aclweb.org/anthology/D/D14/D14-1152.pdf" target=_blank>paper</a>]
</li>
</ul>
